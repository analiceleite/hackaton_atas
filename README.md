## :page_with_curl: Context

This application was built during the Hackathon proposed by Marcelo Mattoso from Robert Bosch GmbH's.<br>
The application receives .zip & .pdf files as inputs and uses Artificial Intelligence to collect relevant data such as employee's name and payment methods.  

## :paperclip: Functionalities

- Reads meeting notes formatted as .pdf files.<br>
- Reads audio logs inside .zip files.<br>
- Transcribes the audio logs and meeting notes.<br>
- Uses Artificial Intelligence to search for meaningful data by command prompt and extracts them.<br>
- The prompt parameters are set as employee name, service evaluation, payment methods and if the purchase was completed.<br>
- All collected data will be listed to the user.
